# Electron-Tailwind-React

## Instalação
```console
$ npm install
$ yarn install
```

## Build
Windows
```console
$ yarn electron:package:win
```
macOS
```console
$ yarn electron:package:mac
```
Linux
```console
$ yarn electron:package:linux
```

## Desenvolvimento
ReactJS
```
npm start
```
ElectronJS
```
npm run electron
```
